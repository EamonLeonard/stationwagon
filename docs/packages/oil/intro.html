Oil utility

The Oil utility is a special package command can be used in several ways to facilitate quick development, help with testing your application and for running Tasks.

Oil is an optional command line utility designed to help speed up development by providing several functions:

  • Introduction
  • Generate
  • Refine
  • Package - Install, update and remove packages.
  • Test - Run Octane test files and see results.
  • Console - Test your code in real time using an interactive shell.

The above is just an example of how you might use it, not how you have to use it. Read on in the next section to understand which methods are part of the generic interface and which will be driver specific: Basic Usage

Configuration

Oil can be configured in a config/Oil.php file and takes the following keys:

Param Type Default Description
driver string|array 
array('simpleOil')
 Login drivers to load, the first will also be the default returned by Oil::instance().
verify_multiple_logins bool 
false
 Whether checking for login continues after one driver has validated a login succesfully, this makes it possible to login in multiple ways at the same time.
